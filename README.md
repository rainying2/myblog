# myblog

> A Vue.js project

**个人博客项目       计算机174班   江霖颖   201710214913**

------

- 前端：vue+vue-router+axios
- 后端：node.js
- 数据库：mysql

## Build Setup

``` bash
# 安装前端依赖
npm install

# 运行前端，端口：localhost:8080
npm run dev

# 安装后端依赖
cd server
npm install

# 初始化和建立数据库
cd server
node database.js

# 运行后端
cd server
node app.js
```
